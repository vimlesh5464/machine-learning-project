# Student Performance EDA

This project performs **Exploratory Data Analysis (EDA)** on a student performance dataset using:

- NumPy
- Pandas
- Matplotlib
- Seaborn

## Steps

1. Load dataset
2. Clean data (dropna, fillna, remove duplicates)
3. NumPy-based statistics: mean, std, correlation
4. Visualization:
   - Histogram of Hours Studied
   - Scatter plot: Hours vs Score (Pass/Fail)
   - Boxplots: Salary & Performance
   - Heatmap of correlations

## Folder Structure

- data → raw dataset (students.csv)
- notebooks → Jupyter notebook for EDA
- outputs → cleaned data & plots
